ADD_EVENT_DISPATCHER()

This function Implements the event dispatcher functions on any object you send in. YOu can call this inside a class you create so all instances of that class are able to dispatchEvents. This is how is currently set up on the JSON_Loader, TimeLine and DragTouch class.
